    ```
    class PipelineOrchestrator:
        def __init__(self, scanner, research, predictor, decision_pipeline,
                     executor, fill_tracker, order_repo, settings, session_factory):
            self._scanner = scanner         # MarketScanner
            self._research = research       # ResearchPipeline
            self._predictor = predictor     # ProbabilityPipeline
            self._decision = decision_pipeline  # DecisionPipeline
            self._executor = executor       # OrderExecutorProtocol
            self._fill_tracker = fill_tracker  # FillTracker
            self._repo = order_repo         # OrderRepository
            self._settings = settings       # Settings
            self._session_factory = session_factory
            self._price_event_queue = asyncio.Queue()
            self._last_predictions = []     # Cache for WS re-eval
            self._last_candidates = []

        async def run(self, stop_event: asyncio.Event) -> None:
            # asyncio.gather three coroutines:
            # 1. _full_cycle_loop(stop_event)
            # 2. _ws_reeval_loop(stop_event)
            # 3. self._fill_tracker.run(stop_event)
    ```

    **_full_cycle_loop(stop_event):**
    - While not stop_event.is_set():
      - Call `_run_full_cycle()`
      - Wait for either stop_event or scan_interval_seconds timeout (using `asyncio.wait_for(stop_event.wait(), timeout=settings.scan_interval_seconds)` with `except asyncio.TimeoutError: pass`)

    **_run_full_cycle():**
    - Generate cycle_id via `str(uuid.uuid4())`
    - Bind cycle_id to logger
    - Stage 1 (Scanner): `await asyncio.wait_for(self._scanner.run_cycle(), timeout=self._settings.stage_timeout_seconds)` — on failure, log error and return (cycle aborted)
    - Extract candidates from scan_result.candidates — if empty, log and return
    - Stage 2 (Research): `await asyncio.wait_for(self._research.run(candidates, cycle_id), timeout=self._settings.stage_timeout_seconds)` — on failure, log and return
    - Stage 3 (Prediction): `await asyncio.wait_for(self._predictor.predict_all(candidates, signal_bundles), timeout=self._settings.stage_timeout_seconds)` — on failure, log and return
    - Cache predictions and candidates for WS re-eval: `self._last_predictions = predictions; self._last_candidates = candidates`
    - Stage 4 (Decision): `await self._decision.evaluate(predictions, candidates)` — no timeout needed (fast, pure computation)
    - Stage 5 (Execute): For each approved decision, call `_execute_decision(decision, log)`
    - Add Prometheus metrics: `CYCLE_TOTAL = Counter("pmtb_pipeline_cycles_total", "Total pipeline cycles completed")`, `CYCLE_DURATION = Histogram("pmtb_pipeline_cycle_duration_seconds", "Full cycle duration")`

    **_execute_decision(decision, log):**
    - Check halt flag: `async with self._session_factory() as session:` query TradingState where key="trading_halted". If value=="true", log warning and return.
    - Compute limit price: `price = int(decision.p_market * 100) + self._settings.price_offset_cents` (Kalshi uses cents). Clamp to 1-99 range.
    - Place order: `result = await self._executor.place_order(market_ticker=decision.ticker, side=decision.side, quantity=decision.quantity, price=price)`
    - Persist to DB: `await self._repo.create_order(market_ticker=decision.ticker, side=decision.side, quantity=decision.quantity, price=price, kalshi_order_id=result.get("order_id"), is_paper=(self._settings.trading_mode == "paper"))`
    - Log order placed with all relevant fields

    **_ws_reeval_loop(stop_event):**
    - While not stop_event.is_set():
      - Wait for event from `self._price_event_queue` with 1-second timeout
      - If event received and `self._last_predictions` is non-empty:
        - Re-run decision pipeline: `await self._decision.evaluate(self._last_predictions, self._last_candidates)`
        - Execute any newly approved decisions
      - Continue on TimeoutError

    **feed_price_event(event: dict) -> None:**
    - Public method for WS price-change handler to inject events into the queue
    - `self._price_event_queue.put_nowait(event)`

    - Test scanner failure: mock scanner raises Exception — verify cycle aborted, no downstream calls
    - Test no candidates: mock scanner returns empty list — verify no research/prediction calls
    - Test halt flag: mock TradingState query returns "true" — verify executor.place_order NOT called
    - Test WS re-eval: put event in queue, verify decision.evaluate called with cached predictions
    - All mocks use `unittest.mock.AsyncMock`
