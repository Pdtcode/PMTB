    - `size(self, decision: TradeDecision) -> TradeDecision`:
      - Assert decision.approved is True (defensive check)
      - Compute b = (1.0 - decision.p_market) / decision.p_market
      - Compute q = 1.0 - decision.p_model

      - Cap: f = min(f, max_single_bet) (SIZE-03)
      - Dollar amount = f * portfolio_value
      - Quantity = max(1, int(dollar_amount)) — floor, never over-size; minimum 1 contract
      - Return updated copy via model_copy(update={quantity, kelly_f=f})
