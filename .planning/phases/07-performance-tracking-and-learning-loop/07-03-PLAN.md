<objective>
Build the LearningLoop that detects trade resolutions via Kalshi settlement polling, feeds resolved outcomes back into XGBoost retraining with recency weighting, and triggers retraining on both a periodic schedule (configurable, default weekly) and Brier score degradation. The retrained model only replaces the live model if hold-out Brier improves.

Purpose: Closes the feedback loop — the system learns from its mistakes and adapts to market regime changes. Without this, the XGBoost model degrades over time as market dynamics shift.

Output: `src/pmtb/performance/learning_loop.py` with full test coverage, plus XGBoostPredictor.train() updated to accept sample_weight.
</objective>
