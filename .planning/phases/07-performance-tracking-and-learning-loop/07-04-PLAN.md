<objective>
Build the BacktestEngine that replays historical trades through the same ProbabilityPipeline and KellySizer code paths used in live trading, with a BacktestDataSource protocol that enforces temporal integrity (no lookahead bias). Results are persisted to the BacktestRun DB table.

Purpose: Strategy changes cannot be deployed safely without backtesting. The backtester validates that model or parameter changes would have improved past performance before they go live. Using the same code paths (PERF-08) ensures backtest results are representative of live behavior.

Output: `src/pmtb/performance/backtester.py` with full test coverage.

       - `BacktestDataSource` class (not Protocol — concrete implementation):
         - Constructor: `__init__(self, session_factory)`
         - `async get_signals(self, market_id: UUID, as_of: datetime) -> list[Signal]` — SELECT * FROM signals WHERE market_id = :mid AND created_at <= :as_of ORDER BY created_at DESC
         - `async get_market_snapshot(self, ticker: str, as_of: datetime) -> dict` — SELECT from markets table, reconstruct MarketCandidate-compatible dict. Use the most recent orderbook snapshot (from ModelOutput.p_market or Market table) as of timestamp.
         - `async build_signal_bundle(self, ticker: str, market_id: UUID, as_of: datetime, cycle_id: str) -> SignalBundle` — builds SignalBundle from temporal-filtered signals
       - `BacktestEngine` class:
         - Constructor: `__init__(self, predictor: ProbabilityPipeline, edge_detector: EdgeDetector, sizer: KellySizer, data_source: BacktestDataSource, session_factory, settings: Settings)`
         - `async run(self, start_date: datetime, end_date: datetime, parameters: dict | None = None) -> BacktestResult` — main backtest loop:
           1. Query resolved trades in [start_date, end_date] ordered by resolved_at ASC
           2. For each trade: use data_source to build MarketCandidate and SignalBundle as-of trade.created_at
           3. Call predictor.predict_one(market, bundle) — SAME code path as live
           4. Call edge_detector.evaluate(prediction.p_model, market.implied_probability)
           5. If edge detected: call sizer.size(decision) — SAME code path as live
           6. Record simulated trade result (would have traded yes/no, actual outcome)
           7. Compute Brier score, Sharpe ratio, win rate, profit factor from simulated results
           8. Return BacktestResult
         - `async persist_result(self, result: BacktestResult) -> None` — write BacktestRun row to DB
         - `async run_and_persist(self, start_date, end_date, parameters=None) -> BacktestResult` — calls run() then persist_result()
