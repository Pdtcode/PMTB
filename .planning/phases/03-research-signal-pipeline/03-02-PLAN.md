    - classify("Terrible loss. Everything crashed.") returns bearish without Claude call (VADER compound < -threshold)
    - classify("The committee met today.") escalates to Claude (ambiguous — abs compound < threshold)
    - When anthropic_api_key is None, ambiguous text returns neutral with confidence=abs(compound) instead of calling Claude
    - confidence is always 0.0-1.0 (abs(compound) for VADER, Claude-returned for escalation)

  <action>
    Create `src/pmtb/research/sentiment.py` with class `SentimentClassifier`:
    - `__init__(self, escalation_threshold: float = 0.3, anthropic_api_key: str | None = None, model: str = "claude-3-5-haiku-latest")`
    - Store a `SentimentIntensityAnalyzer()` instance and optionally an `AsyncAnthropic` client (only if key provided)
    - `async def classify(self, text: str) -> SignalClassification`:
      1. Run VADER `polarity_scores(text)` to get compound
      2. If `compound >= escalation_threshold`: return bullish, confidence=compound
      3. If `compound <= -escalation_threshold`: return bearish, confidence=abs(compound)
      4. Else (ambiguous band):
         - If no Claude client: return neutral, confidence=abs(compound), reasoning=None
         - If Claude client: call `messages.create()` with prompt asking for JSON {sentiment, confidence, reasoning}. Parse response with `json.loads()`. Return SignalClassification from parsed data.
    - Log escalation decisions with loguru `.bind(compound=compound, escalated=bool)`
    - Add Prometheus counter `SENTIMENT_ESCALATIONS` for Claude escalation rate tracking

    - Test clear bearish → no Claude call
    - Test ambiguous → Claude called (mock AsyncAnthropic, return valid JSON)
    - Test ambiguous with no API key → returns neutral without error
    - Test confidence is always 0.0-1.0
