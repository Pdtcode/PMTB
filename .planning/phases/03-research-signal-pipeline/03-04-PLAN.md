  <name>Task 1: ResearchPipeline orchestrator with parallel execution and graceful degradation</name>
  <files>src/pmtb/research/pipeline.py, tests/research/test_pipeline.py</files>
  <behavior>
    - pipeline.run(candidates, cycle_id) runs all 4 agents concurrently for each candidate
    - Parallel execution completes faster than sequential (timing test with mocked agents using asyncio.sleep)
    - When one agent raises an exception, the other 3 results are preserved — SignalBundle has None for failed source
    - When one agent times out (exceeds research_agent_timeout), pipeline continues — timed out source is None
    - Failed/timed-out sources produce None SourceSummary, NOT neutral sentiment
    - Signal rows are written to DB for each successful agent's classifications
    - SignalBundle per market per cycle has correct per-source SourceSummary aggregation
    - pipeline.run returns list[SignalBundle] matching input candidates
  </behavior>
  <action>
    Create `src/pmtb/research/pipeline.py` with class `ResearchPipeline`:

    - `__init__(self, agents: list[ResearchAgent], query_constructor: QueryConstructor, session_factory, timeout: float = 30.0)`:
      - Store agents, query_constructor, session_factory, timeout
      - Build a dict mapping agent.source_name -> agent for lookup

    - `async def _run_agent_safe(self, agent: ResearchAgent, candidate: MarketCandidate, query: str) -> SourceResult | None`:
      - Wrap agent.fetch() in `async with asyncio.timeout(self.timeout):`
      - On TimeoutError: log warning with loguru `.bind(source=agent.source_name, ticker=candidate.ticker)`, return None
      - On any Exception: log exception with loguru, return None
      - On success: return SourceResult

    - `def _aggregate_source(self, result: SourceResult | None) -> SourceSummary | None`:
      - If result is None, return None
      - If result.signals is empty, return SourceSummary(sentiment=None, confidence=None, signal_count=0)
      - Aggregate: majority sentiment, mean confidence, signal_count=len(signals)

    - `async def _resolve_market_id(self, ticker: str) -> uuid.UUID | None`:
      - Query markets table: `SELECT id FROM markets WHERE ticker = :ticker`
      - Return UUID if found, None if market not in DB (log warning)

    - `async def _persist_signals(self, market_id: uuid.UUID, result: SourceResult, cycle_id: str)`:
      - For each SignalClassification in result.signals:
        - Create Signal DB row with market_id, source=result.source, sentiment, confidence, raw_data (include reasoning if present), cycle_id
        - Write via async session using get_session(self.session_factory)
      - Commit after all signals for this source

    - `async def run(self, candidates: list[MarketCandidate], cycle_id: str) -> list[SignalBundle]`:
      - For each candidate:
        1. Build query via self.query_constructor.build_query(candidate)
        2. Resolve market_id via self._resolve_market_id(candidate.ticker) — skip persistence if None
        3. Fire all agents concurrently: `results = await asyncio.gather(*[self._run_agent_safe(agent, candidate, query) for agent in self.agents])`
        4. For each non-None result (and if market_id resolved): persist signals to DB
        4. Aggregate each result into SourceSummary
        5. Build SignalBundle(ticker=candidate.ticker, cycle_id=cycle_id, reddit=..., rss=..., trends=..., twitter=...) — map by source_name
      - Return list of SignalBundles
      - Log cycle summary: total candidates, signals collected, sources failed

    - Add Prometheus metrics:
      - `RESEARCH_SIGNALS_COLLECTED` counter (labels: source)
      - `RESEARCH_AGENT_FAILURES` counter (labels: source, reason=timeout|error)
      - `RESEARCH_CYCLE_DURATION` histogram

    Create `tests/research/test_pipeline.py`:
    - **test_parallel_execution**: Create 4 mock agents that each `await asyncio.sleep(0.1)`. Assert total time < 0.3s (parallel) vs 0.4s (sequential). This proves concurrency.
    - **test_graceful_degradation**: Create 4 agents where one raises RuntimeError. Assert pipeline returns SignalBundle with 3 populated sources and 1 None.
    - **test_timeout_handling**: Create agent with `asyncio.sleep(100)`. Set timeout=0.01. Assert pipeline returns SignalBundle with that source as None.
    - **test_failed_source_is_none_not_neutral**: Assert failed source's SourceSummary is None, not SourceSummary(sentiment="neutral", ...).
    - **test_signal_persistence**: Mock session factory. Assert Signal objects are added to session with correct fields.
    - **test_bundle_assembly**: Assert SignalBundle has correct per-source mapping based on agent source_names.

    Note: DB persistence tests use mock session (AsyncMock for session.add and session.commit) — no real DB needed.
